# Internal Workflow

This document outlines the workflow for using the Drone Footage Merge Tool.

1.  **Preparation**:
    *   Insert drone memory cards into the computer (or card reader).
    *   Ensure all cards are mounted.

2.  **Launch App**:
    *   Open `DroneMerge`.

3.  **Select Folders**:
    *   **Input Root**: Select the folder containing the mounted cards (e.g., `D:\DroneShoot`).
    *   **Output Folder**: Select the destination drive/folder (e.g., `E:\ProjectX\RawFooage`).

4.  **Scan**:
    *   Click "Scan".
    *   Review the summary. Ensure all `VID_` folders are detected.

5.  **Confirm**:
    *   Check "Verify Input".
    *   Check "Verify Output".
    *   Check "I understand files will be merged".

6.  **Merge**:
    *   Click "Start Merge".
    *   Wait for the progress bar to complete.

7.  **Verify**:
    *   Check the completion report.
    *   Open the output folder and spot check a few files.
