# Standard Operating Procedure (SOP)

**For Staff Use Only**

## Pre-Merge Checklist
- [ ] All SD cards are accounted for.
- [ ] Destination drive has enough space (min 500GB free recommended).
- [ ] No other heavy file operations are running.

## Merging Process
1.  Run `DroneMerge`.
2.  Select input parent folder (where all card folders live).
3.  Select output destination.
4.  **Confirm Scan**: Check that the `Total Files` and `Total VIDs` match your shoot notes.
5.  **Run Merge**: Do not interrupt the process.

## Post-Merge
- [ ] Check the `_merge_log_....txt` file in the output folder.
- [ ] Look for any "ERROR" lines.
- [ ] If clear, you may proceed to import into editing software.
